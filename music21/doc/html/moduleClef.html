Class Clef¶

class music21.clef.Clef¶

Inherits from: base.Music21Object (of module music21.base)

groups¶

sign¶
mx¶

Given a music21 Clef object, return a MusicXML Clef object. This might be moved only into PitchClef.

>>> b = GClef()
>>> a = b.mx
>>> a.get('sign')
'G'
>>> b = Treble8vbClef()
>>> b.octaveChange
-1
>>> a = b.mx
>>> a.get('sign')
'G'
>>> a.get('clefOctaveChange')
-1
