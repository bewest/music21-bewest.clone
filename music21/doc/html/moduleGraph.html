music21.graph¶

music21.graph.plotStream(streamObj)¶

Given a stream and any keyword configuration arguments, create and display a plot. Note: plots requires matplotib to be installed. Plot method can be specified as a second argument or by keyword. Available plots include the following: pitchSpace (music21.graph.PlotPitchSpace) pitchClass (music21.graph.PlotPitchClass) quarterLength (music21.graph.PlotQuarterLength) scatterPitchSpaceQuarterLength (music21.graph.PlotScatterPitchSpaceQuarterLength) scatterPitchClassQuarterLength (music21.graph.PlotScatterPitchClassQuarterLength) scatterPitchClassOffset (‘graph.PlotScatterPitchClassOffset) pitchClassOffset (music21.graph.PlotPitchSpaceOffset) pitchSpaceOffset (music21.graph.PlotPitchClassOffset) pitchSpaceQuarterLengthCount (music21.graph.PlotPitchSpaceQuarterLengthCount) pitchClassQuarterLengthCount (music21.graph.PlotPitchClassQuarterLengthCount)

class Graph¶

class music21.graph.Graph¶

An object representing a graph or plot, automating the creation and configuration of this graph in matplotlib. Numerous keyword arguments can be provided for configuration: alpha, colorBackgroundData, colorBackgroundFigure, colorGrid, title, doneAction, figureSize, colors, tickFontSize, titleFontSize, labelFontSize, fontFamily. The doneAction determines what happens after graph processing: either write a file (‘write’), open an interactive GUI browser (‘show’) or None (do processing but do not write output.

Numerous horizontal bars in discrete channels, where bars can be incomplete and/or overlap. Data provided is a list of pairs, where the first value becomes the key, the second value is a list of x-start, x-end points.
