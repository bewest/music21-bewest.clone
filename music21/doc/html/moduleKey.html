music21.key¶

music21.key.keyFromString(strKey)¶
music21.key.sharpsToPitch(sharpCount)¶

Given a number a positive/negative number of sharps, return a Pitch object set to the appropriate major key value.

>>> sharpsToPitch(1)
G
>>> sharpsToPitch(1)
G
>>> sharpsToPitch(2)
D
>>> sharpsToPitch(-2)
B-
>>> sharpsToPitch(-6)
G-
>>> sharpsToPitch(6)
F#
